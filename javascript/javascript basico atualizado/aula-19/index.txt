objetos 

é uma estrutura para trabalhar com dicionarios chave-valor 

a chave é o nome da propriedade 


valor : o que esta armazenado dentro da propriedade 

os elementos nao possuem sequencia. 

uma propriedade pode ter strings, numeros e booleanos. 

pode armazenar qualquer tipo de dados. 

criado com chaves; 

suas funções pode ser referenciada com ponto.
